# Blobcade

**Build. Play. Together.** — a free, open browser game platform and engine: friendly
blocky worlds with AAA-style lighting. Type one command, get a portal of multiplayer
games, and create your own in ~25 lines of TypeScript.

No game assets, no accounts, no build pipeline for creators. **Every model, face,
sound and sky in Blobcade is generated by code at runtime.**
